# Архитектура мобильного приложения ЗАЛОГ.ПРО (iOS / Android)

Приложение спроектировано как гибридная экосистема для двух типов пользователей: Инвесторов и Заемщиков.

---

## 1. Общие экраны (Onboarding & Auth)
1.  **Splash Screen:** Анимированный логотип (золото на антраците).
2.  **Smart Onboarding:** 3 слайда: "Быстрые деньги для бизнеса", "Доходность до 30% под залог", "Безопасность по ФЗ-259".
3.  **Auth Screen:** Вход по номеру телефона + кнопка "Войти через Госуслуги (ЕСИА)".
4.  **FaceID / PIN:** Настройка быстрого входа и биометрии.

---

## 2. Раздел Инвестора (Pocket Investor)
**Цель:** Мониторинг портфеля и быстрые инвестиции.

1.  **Home (Dashboard):**
    *   Виджет общего баланса с live-счетчиком прибыли.
    *   Мини-график доходности (Sparkline).
    *   Кнопка "Пополнить".
2.  **Marketplace (Лента сделок):**
    *   Вертикальный скролл карточек объектов.
    *   Фильтры: Регион, Доходность, LTV, Срок.
    *   Кнопка "AI-Аудит" на каждой карточке.
3.  **Deal Detail (Карточка объекта):**
    *   Галерея фото и live-видео.
    *   Результаты ИИ-скоринга (Safety Score).
    *   Калькулятор прибыли (ползунок).
    *   Кнопка "Инвестировать".
4.  **Portfolio:**
    *   Список активных долей.
    *   История начислений (транзакции).
    *   График распределения рисков (Pie chart).

---

## 3. Раздел Заемщика (Fast Loan)
**Цель:** Оформление залога и контроль графика платежей.

1.  **Home (Status):**
    *   Этап текущей заявки (Timeline).
    *   Сумма одобренного лимита.
2.  **Collateral Wizard (Сбор данных):**
    *   Шаг 1: Адрес и тип объекта (автоподбор по кадастру).
    *   Шаг 2: Встроенная камера для LIVE-съемки (не дает загружать из галереи).
    *   Шаг 3: Liveness check (поверните голову, улыбнитесь).
3.  **Legal Hub:**
    *   Просмотр сформированного договора.
    *   Кнопка "Подписать через Госключ / СМС".
4.  **Payment Schedule:**
    *   Календарь предстоящих платежей.
    *   Кнопка "Оплатить" (Apple Pay / СБП).

---

## 4. Специфические мобильные функции (Mobile-Only)
*   **Push-уведомления:** Мгновенно о новых лотах и о зачислении процентов.
*   **Live-Чат:** Поддержка 24/7 с ИИ-ассистентом.
*   **AR-Оценка:** Инструмент для фото-фиксации дефектов залога с наложением GPS и времени.
*   **Offline Cache:** Просмотр портфеля и документов без доступа к сети.

---

## Технологические рекомендации
*   **Stack:** Flutter (для идеальной плавности анимаций графиков).
*   **Backend Sync:** REST API + WebSocket (для живых графиков прибыли).
*   **Security:** Шифрование данных AES-256, SSL Pinning.
